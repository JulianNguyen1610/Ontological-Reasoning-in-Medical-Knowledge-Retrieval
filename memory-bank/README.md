# Memory Bank

This folder is a lightweight project memory bank for `MedicalRetrieval`.

Use it to get up to speed quickly before editing code:

- `00-project-overview.md`: scope, goals, inputs, outputs
- `01-architecture.md`: main modules and responsibilities
- `02-workflows.md`: setup, run, test, and data flow
- `03-decisions-and-risks.md`: assumptions, risks, and follow-up items
- `04-medlink-ie-context.md`: fixed context and canonical MedLink-IE documents

Suggested reading order for new contributors:

1. `00-project-overview.md`
2. `01-architecture.md`
3. `02-workflows.md`
4. `03-decisions-and-risks.md`
5. `04-medlink-ie-context.md`, followed by the five canonical files listed there when working on MedLink-IE
