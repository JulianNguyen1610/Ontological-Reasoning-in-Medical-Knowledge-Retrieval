# MedLink-IE Fixed Project Context

MedLink-IE v2 is the medical information extraction framework maintained at the
repository root. The following canonical documents are fixed project context and
must be consulted before changing the framework, its contracts, or its pipeline:

- `README.md` — project scope and layout.
- `docs/MedLink-IE_v2_Framework.md` — end-to-end architecture and implementation specification.
- `specs/TASK_CONTRACT.md` — input, output, span, assertion, candidate, and scoring contracts.
- `specs/ANNOTATION_GUIDE.md` — entity boundary, type, assertion, overlap, and adjudication policy.
- `specs/terminology_manifest.yaml` — terminology provenance, normalization, and index artifact contract.

These files are the source of truth for the MedLink-IE project. The requested
path string `README.mddocs/...` was interpreted as the two separate files
`README.md` and `docs/MedLink-IE_v2_Framework.md`.
